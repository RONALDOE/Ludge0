## Frontend

---

### Installation

Clone the [repository](https://github.com/Ludgeo/ludgeo.git) and `execute` the following commands to install all the dependencies:

```bash
$ cd frontend
$ npm install
```

### Initialization

To start the project `execute` the following commands:

```bash
# Development
$ npm run dev

# Production Preview
$ npm run build
$ npm run preview
```
